# Fuente de datos

## AEMET OpenData

AEMET OpenData será la fuente principal del proyecto.

## Datos que queremos obtener

- Temperatura máxima.
- Temperatura mínima.
- Temperatura media.
- Precipitación.
- Velocidad del viento.
- Humedad.
- Presión.
- Información de estaciones meteorológicas.
